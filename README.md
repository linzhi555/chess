#### 使用说明

![](./gamming.png)

TAB : switch input mode and board mode

input mode : 

    Enter: input buffer

board mode:

- Up:   move up

-  Left: move left

- Right: move right

- Down: move down

- Enter :  First Enter is select , Second Enter is confirm
